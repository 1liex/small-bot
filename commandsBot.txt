أولاً: أوامر التشغيل (Run Commands):

جوجل ← لفتح متصفح جوجل كروم.

الملاحظات ← لفتح برنامج المفكرة (Notepad).

بريف ← لفتح متصفح Brave.

في اس كود ← لفتح برنامج VS Code.

تاسك مانجر ← لفتح مدير المهام.

انستقرام ← لفتح موقع إنستقرام.

تيك توك ← لفتح موقع تيك توك.

فاصل ← لفتح موقع فاصل بلس.

جوجل اي اي ← لفتح Gemini AI.

سي ام دي ← لفتح موجه الأوامر (Command Prompt).

شات جي بي تي ← لفتح ChatGPT.

ديسكورد ← لفتح تطبيق ديسكورد.

يوتيوب ← لفتح موقع يوتيوب.

بلاي ليست ← لفتح قائمة تشغيل يوتيوب محددة.

مونجو ← لفتح برنامج MongoDB Compass.

مترجم ← لفتح ترجمة جوجل.

بلاك بورد ← لفتح نظام البلاك بورد بجامعة طيبة.

جامعه ← لفتح بوابة القبول/التسجيل بجامعة طيبة.

واتساب ← لفتح تطبيق واتساب.

ستيل سيريس ← لفتح برنامج SteelSeries GG.

كاميرا ← لفتح برنامج Iriun Webcam.

ايموجي ← لفتح معرض الإيموجي.

او بي اس ← لفتح برنامج OBS Studio.

ستريم ديك ← لفتح برنامج Stream Deck.

سطر ← لفتح منصة سطر التعليمية.

جيت هوب ← لفتح موقع GitHub.

لينكد ان ← لفتح موقع LinkedIn.
